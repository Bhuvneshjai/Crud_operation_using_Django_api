1. Install Django:

2. Run pip install django in your terminal/command prompt

3. Create a new Django project:
Run django-admin startproject my_project_api in your terminal/command prompt

4. Create a new Django app:
Run python manage.py startapp product_listing in your terminal/command prompt

5. Define the model for the product:
Open product_listing/models.py and add code

6. Migrate the database:
Run python manage.py makemigrations in your terminal/command prompt
Then run python manage.py migrate in your terminal/command prompt.

7. Create a serializer for the product model:

8. Create a new file product_listing/serializers.py and add code

9. Create views for the product:
Open product_listing/views.py and add code

10. Create URLs for the views:
Open product_listing/urls.py and add code

11. Add the app URLs to the project's URLs:
Open my_project_api/urls.py and add code

12. Test the API:
Run python manage.py runserver in your terminal/command prompt.
Open your web browser and go to http://localhost:8000/api/products/ to view the product listing API.
